This project is an AI-powered Notes Generator that helps users create summarized notes from input text using Generative AI. It simplifies studying by converting large content into short, easy to understand notes.

Features:

- Generate notes automatically
- Simple and user friendly interface
- Fast processing of input text
- Clean and responsive UI

Tech Stack:

- HTML
- CSS
- JavaScript
- Node.js
- OpenAI API

Project Structure:

- public/ → frontend files
- server.js → backend logic
- package.json → dependencies
- .env.example → API key format

Conclusion:

This project shows how Generative AI can be used to automatically generate notes and improve learning efficiency.
